
 Projeto Calculadora (Shell Script + Python)

Projeto desenvolvido para praticar lógica de programação, Shell Script e Python, com versionamento no GitHub.

---

 Objetivo

Criar uma calculadora que recebe dois números, permite escolher uma operação (+, -, *, /) e retorna o resultado.

---

 Estrutura

calculadora.sh
calculadora.py
README.md



Como executar (Shell)

chmod +x calculadora.sh
./calculadora.sh

---

 Como executar (Python)

python3 calculadora.py

---

 Funcionamento

* Solicita dois números
* Solicita a operação
* Calcula e exibe o resultado


 Exemplo

Digite o primeiro número: 10
Digite o segundo número: 5
Operação: *

Resultado: 50



Tecnologias

Shell Script (Bash)
Python
